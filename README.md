## Execise 1

I used Laravel & livewire and tailwindcss.

### Requirement

- Php 7.4^
- node.js 10^
- mySql

### Tools 

- Laravel v8^
- Livewire v2.6
- Tailwind 

### Install the project

- composer install
- npm install or yarn if you prefer
- run the migration
- run the php artisan db:seed to populate dummy data
- run test php artisan test
